# DevTools Platform

무료 개발자 도구를 한눈에 찾아볼 수 있는 플랫폼입니다.

## 주요 기능

- 🔍 **실시간 검색** - Fuse.js 기반的高速 검색
- 🏷️ **카테고리 필터** - AI/ML, CI/CD, Database, API 등
- ⭐ **북마크** - 즐겨 찾는 서비스 저장
- 📱 **반응형 디자인** - 모바일/데스크탑 지원

## 시작하기

1. `index.html` 파일을 브라우저에서 열거나
2. GitHub Pages에 배포하세요

## GitHub Pages 배포

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/your-username/dev-tools-platform.git
git push -u origin main
```

Settings > Pages > Source: main branch

## 데이터 업데이트

`services.json` 파일을 수정하여 서비스를 추가/삭제할 수 있습니다.
